# Sistema de Estoque

Projeto de sistema de estoque desenvolvido com **Spring Boot**.

## Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Persistência em banco H2

## Como executar
1. Clone o repositório
2. Abra no IntelliJ
3. Execute a classe `SistemaEstoqueApplication`
4. Acesse:
    - API: `http://localhost:8080/produtos`
    - H2: `http://localhost:8080/h2-console`
